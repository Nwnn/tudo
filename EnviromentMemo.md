# エディタ VSCode
https://code.visualstudio.com/

1. 左端の四角っぽいアイコン（拡張機能）を選択
2. japanと入力
3. Japanese Language Pack for Visual Studio Codeを導入
4. 再起動

# Node.jsの導入
https://nodejs.org/ja/
LTS、最新版どちらでも構わない

導入が終わったらコマンドプロンプト(スタートメニュー、cmdを入力)を開いて
```
node -v
```
